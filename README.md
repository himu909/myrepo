## How to Use

1. **Open the Project**
   - Open `index.html` in your web browser
   - You can clone this repository and open the file locally
   - Or view it online if hosted on GitHub Pages

2. **Fill Out the Form**
   - Enter your name
   - Enter your email address
   - Type your message
   - Click "Submit Form"

3. **View Results**
   - Your submitted data will be displayed below the form
   - Each submission includes a timestamp
   - Click "Clear Form" to reset and submit again
